# depsee

Goコードの構造体・関数・インターフェースの依存関係を可視化し、安定度（変更容易度）をMermaid記法で出力するCLIツール

## ディレクトリ構成（雛形）

```
cmd/depsee/           # CLIエントリポイント
internal/analyzer/    # 静的解析ロジック
internal/graph/       # 依存グラフ・安定度算出
internal/output/      # Mermaid出力
testdata/             # サンプルGoコード・テスト用
```
